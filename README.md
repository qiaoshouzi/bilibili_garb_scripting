## 如何使用？

1. 下载 [Scripting](https://apps.apple.com/app/apple-store/id6479691128?pt=126805668&ct=offical_page&mt=8)
2. 点击 [这里](https://scripting.fun/import_scripts?urls=%5B%22https%3A%2F%2Falt-r2.cfm.moe%2Fscripting%2FBili_Garb.zip%22%5D) 导入脚本
3. 点击 哔哩装扮 开始运行脚本
