# 哔哩装扮 Scripting

部分文件来自 `https://github.com/Honye/scripting-scripts`

## 如何使用？

1. 下载 [Scripting](https://apps.apple.com/app/apple-store/id6479691128?pt=126805668&ct=offical_page&mt=8)
2. 点击 [这里](https://scripting.fun/import_scripts?urls=%5B%22https%3A%2F%2Fgithub.com%2Fqiaoshouzi%2Fbilibili_garb_scripting%2Freleases%2Flatest%2Fdownload%2FBili_Garb.zip%22%5D) 导入脚本
3. 点击 哔哩装扮 开始运行脚本
